🎬 Christmas Movie Picker – Power BI Dashboard
📊 Project Overview

This project is a Power BI practice dashboard created to explore and analyze Christmas-themed movies through interactive data visualization. The dashboard allows users to filter movies based on genre, ratings, and runtime, making it easier to discover popular movies and analyze viewer trends.
The goal of this project was to practice data visualization, dashboard design, and interactive reporting using Power BI while building a clean and user-friendly analytical dashboard.

✨ Dashboard Features

✔ Interactive Genre filter to explore movies by category
✔ Ratings filter to analyze movies based on audience ratings
✔ IMDB rating range slider for dynamic filtering
✔ Runtime range filter for selecting movies by duration
✔ Movie dataset table displaying Title, Genre, Director, and Release Year
✔ Top 5 Movies by Votes visualization to highlight the most popular films

📈 Key Insights

Identify the most popular Christmas movies based on votes
Analyze movies across different genres
Explore movie runtime and rating patterns
Discover highly rated Christmas films using interactive filters
